Project Summaries

Motorcycle Project: Used SolidWorks to create an assembly of a motorcycle inspired by motorcycles from Suzuki. Part of this project involved creating relevant exploded views, section views, engineering drawings, and a bill of materials.

Tree Puzzle Project: Used SolidWorks to create a 3D puzzle inspired by the emerald ash tree. The tree falls apart after being unlocked by an emerald ashborer key. This assembly was then 3D printed by first uploading the parts to Z-Suite and putting them in a practical orientation to be printed. In addition, I also used SolidWorks Composer to create a clean and easy-to-follow assembly/instruction manual.
